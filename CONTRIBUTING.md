# Contributing to Auto Academic Paper

Thank you for your interest in contributing to Auto Academic Paper!

## Policy: No External Contributions

To ensure long-term sustainability, strict licensing integrity, and codebase stability, **we are not accepting external code contributions, pull requests, or feature additions at this time.**

This repository is maintained as a stable reference implementation. 

## Reporting Issues

While we cannot accept code changes, we value your feedback. If you encounter a bug or have a suggestion, please feel free to open an issue or contact the maintainer directly.

Thank you for understanding and for using Auto Academic Paper!
